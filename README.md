
# WebPaint

Projeto para fins didáticos - WebPaint para criação de desenhos com responsividade
## Funcionalidades

- Defina cor, espessura do traço e limpe a tela
- Arraste o mouse para pintar


## Stack utilizada

**Front-end:** HTML, CSS, Javascript e Bootstrap
